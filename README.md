# Skill Battle

## 概要

体力の続く限りひたすら現れ続ける敵とバトルしていくドラクエ風バトルゲーム。勝利後の報酬を有効活用し、できるだけ多くの敵を倒すことが目的。  
体力は戦闘後に回復せず、次の戦闘に持ち越しとなる。目の前の敵だけでなく、長い戦いを意識したリソース運用が肝。

## 報酬とは

- ステータスアップ  
  永続的なステータス向上
- 装備取得  
  装備により一時的なステータス向上
- スキル習得  
  戦闘の役に立つ特殊能力の習得
- アイテムの取得
  戦闘の役に立つ道具の取得。強力だが一度使うとなくなる。
- キャラクターチェンジ
  キャラクターをごっそり取り換える。HPおよびMPの回復手段。
